# bluebell
bluebell是一个基于gin框架和Vue框架搭建的前后端分离的web项目。


## 技能清单

1. gin框架
2. zap日志库
3. Viper配置管理
4. swagger生成文档
5. JWT认证
6. 令牌桶限流
7. Go语言操作MySQL
8. Go语言操作Redis
